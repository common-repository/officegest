Create all your verified invoices directly in OfficeGest automatically.


== Description ==
This plugin is used to have a full online store synced with OfficeGest.
OfficeGest is an invoicing cloud based software with stock management features.
This plugin required a valid officegest invoice cloud account with access to API.
With this plugins you will be able to:

*  Sync your products to Woocommerce
*  Create all your invoices automatically when a order is paid
*  Nif for Portugal
*  Automatic Fields for Brand and Ean in Single Articles
*  Automatic Sync your Stock
*  Sell AutoParts

Manual available in <a href="https://ps.w.org/officegest/assets/manual.pdf"> this link.<a>

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/officegest` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Manual available in <a href="https://ps.w.org/officegest/assets/manual.pdf"> this link.<a>
